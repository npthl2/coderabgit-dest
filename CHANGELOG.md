## [1.0.0] - 2025-06-24

### Added
- Initial release of Simple Utils library
- `reverse_string()` function for string reversal
- `count_words()` function for word counting in sentences
- `celsius_to_fahrenheit()` function for temperature conversion
- Basic module structure and organization

### Features
- Simple, efficient utility functions
- No external dependencies required
- Python 3.6+ compatibility
- Clear and intuitive function names
